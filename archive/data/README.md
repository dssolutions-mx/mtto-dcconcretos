# Archived data files

- **`csv/`** — Expense (`GASTOS*`), diesel, inventory, plant exports, etc. Moved from repository root in 2026.
- **`csv/sync-outputs/`** — Sample outputs from employee sync scripts (formerly `scripts/out/`).
- **`spreadsheets/`** — Excel workbooks (e.g. inventory).

These are **not** loaded by the app; use them for imports, audits, or historical reference. Regenerate fresh exports as needed and avoid committing large new dumps unless intentional.
