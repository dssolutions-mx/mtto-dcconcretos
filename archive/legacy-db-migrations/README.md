# Legacy SQL (archived)

**Not used to apply schema changes.** This project updates the database via **Supabase MCP** and sometimes the **Supabase CLI** (`supabase/migrations/`). The files here are a **historical snapshot** of SQL that was once maintained in-repo (formerly `migrations/sql/` and root `migrations/*.sql`).

- **`sql/`** — former `migrations/sql/*.sql`
- **`root/`** — former root-level files inside `migrations/` (e.g. manual financial adjustments, storage policies)

For the live schema, use the hosted Supabase project, dashboard SQL, or regenerate TypeScript types from the linked project.
